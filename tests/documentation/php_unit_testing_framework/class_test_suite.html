Detailed Description

Collects together test cases and enables them to be run together.

This class collects together a series of test cases.

  • To add a test case call the TestSuite::AddTest with a string containing the name of the class (which should be defined by extending TestCase).
  • To run all the test cases use the TestSuite::Run method.
  • The method TestSuite::AllPassed returns true if all the tests pass.
